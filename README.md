
Olar Canada English Club 🇨🇦

Site estático de aulas de inglês interativas (sem React).

Como rodar:
- Opção 1 (rápida): abra `index.html` no navegador
- Opção 2 (recomendado): rode um servidor local na pasta do projeto:
  - `python3 -m http.server 5173`
  - depois acesse `http://localhost:5173`

Conteúdo (lessons):
- As aulas ficam em arquivos separados em `lessons/` (ex.: `lessons/airport.js`).
- Para adicionar uma nova aula, crie um novo arquivo em `lessons/` e inclua ele em `index.html`.
